# Projet_filtrage

L'interface est je penses relativement intuitive en grande partie, toutefois certain point necessite d'être détaillé ici.

Tout d'abord, pour utiliser l'interface il est nécessaire de chargé un signal, pour cela les boutons :
 "white noise", "pink noise" et "use datagramEEG" peuvent être utiliser selon le  signal que l'on veut analyser.

Ensuite différente valeurs peuvent être modfié par l'utilisateur, mais ce dernier peut toujours revenir au valeur par défault avec le bouton "RESET"

Pour les valeurs de N l'on ne peut pas mettre n'importe quel valeurs. Soit on laisse les valeurs pas défault, soit on indique la borne
minimal et maxamile de l'inteveralle dans lequel N prendra c'est valeurs. En indiqunad le minimum dans Nmin et le Maximum dans Nvalue.
